# Agency.AI - Digital Agency Website

A modern, responsive digital agency website built with **React 19**, **Vite**, and **Tailwind CSS**. Designed with a clean, conversion-focused UI, smooth animations, dark/light mode toggle, and a fully functional contact form.

## 🌟 Features

- **🌙 Dark / Light Mode** with smooth theme transition
- **📱 Fully Responsive** design optimized for all devices
- **⚡ Blazing Fast** performance using Vite
- **🎨 Modern & Elegant UI** built with Tailwind CSS
- **🎬 Smooth Animations** powered by Motion (Framer Motion)
- **🔔 Toast Notifications** using React Hot Toast
- **📩 Functional Contact Form** powered by Web3Forms (no backend required)
- **Trusted Logos Section** & Social Proof
- **Clean Project Structure** and reusable components

## 💼 Services Showcased

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/JOHEAVEN7/Social-Media-Management_Agency.git

# Navigate to project folder
cd Social-Media-Management_Agency

# Install dependencies
npm install

# Start development server
npm run dev
